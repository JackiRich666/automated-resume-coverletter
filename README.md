Инструкция по установке и запуску проекта
Самый первый шаг уровня 0 - переименовать файлы .env.example в .env
Сделать это нужно в папках frontend и backend
Если необходимо, можно изменить порты в тексте файлов

1. Установите необходимое ПО
Перед началом работы убедитесь, что на вашем компьютере установлены следующие программы:

Node.js
Скачайте и установите Node.js (версия 16 или выше) с официального сайта:
https://nodejs.org/
Рекомендуется выбрать LTS-версию. После установки проверьте, что Node.js и npm доступны в командной строке:

node -v
npm -v
Ollama
Скачайте и установите Ollama с официального сайта:
https://ollama.com/
После установки проверьте, что Ollama доступна в командной строке:

ollama --version
2. Установите зависимости проекта
Запустите скрипт modules-node.bat (дважды кликните по файлу).
Этот скрипт установит все необходимые Node.js-пакеты для бэкенда и фронтенда. Дождитесь завершения процесса (в окне командной строки появится сообщение об успешном завершении).

3. Загрузите модель искусственного интеллекта
Запустите скрипт ollama AI download model.bat.
Он скачает модель qwen2.5:14b (около 9 ГБ). Это может занять длительное время в зависимости от скорости вашего интернета. Не закрывайте окно до завершения загрузки.

4. Запустите приложение
Запустите скрипт start-app.bat.
Откроются три командных окна:

Сервер Ollama (фоновая служба)

Бэкенд (API-сервер)

Фронтенд (React-приложение)

Дождитесь, пока во всех окнах появятся сообщения о готовности. Обычно это занимает несколько секунд.

После этого откройте браузер и перейдите по адресу:
http://localhost:3000

Вы увидите главную страницу сервиса.

5. Работа с приложением
Используйте интерфейс для создания резюме, генерации сопроводительных писем и анализа текста. Все запросы к искусственному интеллекту обрабатываются локально через Ollama.

6. Завершение работы
Чтобы остановить приложение, закройте все три командных окна (или нажмите Ctrl+C в каждом из них).

Примечания
Скрипты modules-node.bat и ollama AI download model.bat нужно запускать только один раз после первого скачивания проекта.

При повторных запусках используйте только start-app.bat.

Если вы меняли порты в файлах .env, убедитесь, что они совпадают с настройками в скриптах.
